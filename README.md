# 🤖 Ejercicio 1: Gestión de Proyectos con IA y GitHub
### Módulo de Trazabilidad y Planificación - 3R Automatización

Este repositorio documenta la base estratégica del ecosistema industrial, centrada en la integración de agentes de IA para la gestión de proyectos técnicos y su control de versiones en GitHub.

## 📋 Resumen del Proyecto
Se ha desarrollado la planificación de **"SmartLogistics-AI"**, un sistema inteligente para la optimización de rutas en almacenes automatizados (AGVs). El enfoque principal es demostrar cómo la IA puede asistir no solo en la codificación, sino en la estructuración de la documentación y la gestión de repositorios.

## 🚀 Contenido del Repositorio
- `Documentacion_Ex1.docx`: Informe académico completo siguiendo estándares de rigor técnico.
- `.git`: Historial de versiones que asegura la autoría y el progreso del trabajo.
